# TypeScript
>安裝TypeScript套件
```command
npm i typescript -g
```

>查看版本
```command
tsc --version 
```

>執行TypeScript
```command
/*無法直接編譯TypeScript，需轉成js
 *直接執行語法，會自己轉譯產生javascript檔案
 */
tsc 02DataType.ts & node 02DataType.js   
```

>建立Commpoent元件
```command
ng g c MyInterpolation
```
